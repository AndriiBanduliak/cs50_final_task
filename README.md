# Python Search Engine

This is a simple Python search engine developed as the final project for the CS50 course by Harvard University. The project involves web scraping Python-related questions and descriptions using the `bs4` module and storing them in a MongoDB database. The web application is built using HTML and Jinja templating.

## Features

- Web scraping of Python-related questions and descriptions
- Storage of scraped data in a MongoDB database
- User-friendly web interface for searching and retrieving information

## Technologies Used

- Python
- Beautiful Soup (`bs4`) for web scraping
- MongoDB for data storage
- HTML and Jinja templating for web interface
